# Roadmap — simple-todo-list-app (solo)

## Goal
Ship a minimal todo list app that supports adding, listing, completing, and removing items.

## Scope decision (tiny product)
The product is intentionally ONE tiny self-contained file. The roadmap is therefore a single
milestone with a single implementation task — enough to exercise the full
plan -> build -> review -> integrate loop honestly without gold-plating.

## Milestones
1. **M1 — Build the todo app (single file).**
   - Task T1: Create the todo list app as one self-contained file in the project.
   - Owner: fullstack-developer (set at breakdown).
   - Acceptance lives in `specs/T1.md`.

## Sequence
M1 → Review (CTO covers QA) → Integration (CTO covers DevOps) → Done.

## Coverage notes (solo)
- No qa-engineer active → CTO covers Review/QA (PROTOCOL §7).
- No devops/sre/cloud active → CTO covers Integration (PROTOCOL §7).
